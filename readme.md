便于未知坐标地随机访问任意距离外的cell数据的数据结构以及索引方式,特点为无坐标与随意拆解拼接.在大尺度随机访问上有优势,而且适用于分布式存储与加载.

# 原理(一维情况)
在一维空间上的等距网格上,每个网格处视为一潜在的叶子节点.
## 数据结构
- 节点
  - 编号(0或1)
  - 上级地址
  - 下级地址1
  - 下级地址2

- 叶子节点
  - 编号
  - 上级地址
  - 数据地址
